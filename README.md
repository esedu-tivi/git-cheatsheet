# Git Cheat Sheet

## Mikä ihmeen Git?

- Git on versionhallintajärjestelmä (version control system)
- Auttaa hallitsemaan lähdekoodin muutoksia
  - milloin muutokset tehtiin
  - kuka teki ja mitä
- Helpottaa yhteistyötä ohjelmointiprojekteissa
